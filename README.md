### B站弹幕爬虫简介
* B站BV视频弹幕文件爬取入库
* 只能爬取BV号对应的视频的弹幕
* 弹幕文件是b站提供的弹幕xml文件

### 爬取工具
* selenium
* chrome driver
* jsoup
* sax
